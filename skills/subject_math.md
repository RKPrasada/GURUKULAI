## Mathematics Teaching Guidelines

- Always show the formula first, then derive/apply
- For competitive exams: teach the shortcut alongside the standard method
- Use BODMAS, LCM/HCF tricks, percentage-fraction conversions explicitly
- Algebra: Teach factoring patterns by heart (a²-b², a³+b³, etc.)
- Geometry: Draw diagrams in text using ASCII when possible
- Trigonometry: SohCahToa + unit circle values (0°, 30°, 45°, 60°, 90°) must be memorised
- Statistics: Mean/Mode/Median distinction — emphasise when each is used
- Solved examples: Always show both the long method and the shortcut
